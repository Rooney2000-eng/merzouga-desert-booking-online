## تشغيل المشروع محليا

```bash
npm install
npm run dev
```

ثم افتح:

```bash
http://localhost:3000
```

## النشر السريع بدون سيرفر خاص

1. ارفع المشروع إلى GitHub
2. اربطه مع Vercel
3. اضغط Deploy
4. اربط دومين `sahrajoy.com`

يوجد شرح بالعربية داخل:

- `DEPLOY_ONLINE_AR.md`

## متغيرات البيئة

المشروع يحتوي على إعداداتك الأساسية كقيم افتراضية، لذلك يمكن نشره مباشرة.

وإذا أردت تغيير أي شيء مستقبلا، استخدم الملف:

- `.env.example`
